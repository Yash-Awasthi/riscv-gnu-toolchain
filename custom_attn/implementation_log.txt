 OVERVIEW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Custom RISC-V instruction: attn rd, rs1, rs2
  Mathematical operation:    Attention(Q, K, V) = softmax(Q*K^T / sqrt(d_k)) * V

  Two structs:
    1. attn_dims_t : { rows, cols, seq_len, d_model }
    2. attn_qkv_t  : { float *Q, float *K, float *V }

  The C program calls __builtin_riscv_attn() — NO inline assembly.
  The modified GCC compiler emits the "attn" instruction automatically.

  Added as first entry in riscv_opcodes[]:

  Addition 1 — UNSPEC constant (inside define_c_enum "unspec"):

  Addition 2 — Instruction pattern (at end of file):
    (define_insn "riscv_attn"
      [(set (match_operand:DI 0 "register_operand" "=r")
            (unspec:DI [(match_operand:DI 1 "register_operand" "r")
                        (match_operand:DI 2 "register_operand" "r")]

      ""
      "attn\t%0,%1,%2"
      [(set_attr "type" "arith")
       (set_attr "mode" "DI")])
